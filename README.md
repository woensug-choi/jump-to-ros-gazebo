**로봇 시뮬레이션의 de-facto 벤치마크 기준 프레임워크인 ROS와 Gazebo.**

> * 최상의 가독성을 위해 데스크탑 환경에서 [Chrome](https://www.google.com/chrome/) 또는 [Edge](https://www.microsoft.com/edge) 사용을 권장합니다.
